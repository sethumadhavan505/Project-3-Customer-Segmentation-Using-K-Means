# Customer Segmentation Using K-Means

## Problem Statement
Businesses need to target different customers differently based on their behavior.

## Objective
Cluster customers into meaningful groups using unsupervised learning.

## Dataset
- Age  
- Income  
- Purchase frequency  
- Spending amount  

## Methodology
- Data preprocessing and scaling
- Apply K-Means clustering
- Assign cluster labels
- Visualize customer groups

## Visualizations
- 2D scatter plot showing customer clusters
- Clustered customer dataset

## Output
- `customer_clusters.csv`
- `customer_clusters.png`

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn
